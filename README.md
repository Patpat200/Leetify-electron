# Leetify app

![sample](leetify.png)



## Requirements

- [NodeJS](https://nodejs.org) v20 or +

## Installation

### Generate package
```sh
npm i --save-dev electron
npm install electron-builder --save-dev
npm run build-win

```

### Use on Windows

1. Move the executable folder `dist/Leetify-win32-x64` anywhere you like it

2. Run `Leetify.exe` and enjoy!
